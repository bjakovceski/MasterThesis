INFORMATION!
The datasets used for preparing new datasets that was used in Stanford NER are not included in the CD, because thay are big and will not fit to the size of the CD.
With using the classes from cleaningDatasets package and dataTree package from source code, and downloaded DBpedia NIF dataset, the data used for preparing new datasets ready for Stanford NER can be created.


Content of the CD:

CreateModels folder contains the code implementation that was used in this thesis. It is a MAVEN project. 
Stanford NER folder contains the Stanford NER application, datasets and trained models used in experiments.
Thesis LaTeX folder contains thesis source code in LaTeX
Text folder contains the Diploma thesis in PDF format.
